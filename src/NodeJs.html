Node.js Streams

Streams are object that let you read data from a source or write data to a destination in continous fashion.In Node.js, there are four types of streams;

Readable : Stream which is used for read operation.

Writeable : Stream which is used for write operation.

Duplex : Stream which can be used for both read and write operation.

Transform : A type of duplex stream where the output is computed based on input.

Events used for Streams

Each type of Stream is and EventEmitter instance and throws several events of different instance of times.
Example : some of the commonly used events ar : -

data : This event is fired when there is data is available to read.

end : This Event is fired when there is no more data to read.

error : This Event is fired when there is any error receiving or writing data.

finish : This event is fired when all the data has been flushed to underlying system.

const fs = require("fs");
const http = require("http");
const hostname = "127.0.0.1";
const port = 3000;
const server = http.createServer();
server.on("request", (req, res) => {
  /*   old Method  
    fs.readFile('index.txt','utf-8',(error,data)=>{
           if(error) return console.log(error);
           console.log(data);
           res.end(data);
       });
  */

  const rstream = fs.createReadStream("file.txt");
  // data Events & chunkdata as parameter that data handle
  rstream.on("data", (chunkdata) => {
    res.write(chunkdata);
  });
  // End Data Events
  rstream.on("end", () => {
    res.end();
  });
  // error Events call when file now found or other isssue
  rstream.on("error", () => {
    res.end("File Not Found");
  });
});
//Server listen the port
server.listen(port, hostname, () => {
  console.log(`Server is Running on : http://${hostname}:${port}`);
});

Serving static files in Express

To serve static files such as images, CSS files, and JavaScript files, use the express.static built-in middleware function in Express.

Middleware : is the those layer that b/w request and response.

express.static(root, [options]);

For Example:use the following code to serve images, CSS files, and JavaScript files in a directory named public.You can use multiple static direcory in the express it is depend on you.

app.use(express.static('public'));
//By Default __dirname is root then you can change to according need
const public = path.join(__dirname, '../public');
app.use(express.static(public));
