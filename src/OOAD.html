Inheritance

A process by which a class can extend another class, inheriting all its, variables, methods, while redefining (overriding) some of them, and/or adding its own.

  • The existing class that is reused to create a new class Super class,Base class or Parent class.
  • The new class that inherit the properties and functions of an existing is known as Sub class, Derived Class or Child Clas.
Advantages
  • “Inheritance is the mechanism to provides the power of reusability and extendibility”
  • Inheritance reuse existing class to build new class.

Types of Inheritance

Five types of inheritance;

  • Single inheritance
  • Multi-level inheritance
  • Hierarchical inheritance
  • Hybrid inheritance
  • Multiple inheritance

Single Inheritance : When a class inherits another class, it is known as a single inheritance.

Multi-level inheritance When a class B is inherit from class A then another class c is inherit from B is known as Multi-level inheritance.

Hierarchical inheritance : Hierarchical inheritance is a kind of inheritance where more than one class is inherited from a single parent or base class.
One Base Class but multiple derived class.

Hybrid inheritance : Hybrid Inheritance in Java is a combination of Inheritances. In this type of Inheritance, more than one kind of inheritance is observed.
Example : if we have class A and class B that extend class C and then there is another class D that extends class A, then this type of Inheritance is known as Hybrid Inheritance.

Multiple inheritance : In Multiple Inheritance we used interfaces in java.But in c/c++ we use multiple inheritance.Java programming language supports multiple inheritance of type, which is the ability of a class to implement more than one interface. An object can have multiple types: the type of its own class and the types of all the interfaces that the class implements.

Encapsulation

Hiding "sensitive" data from user.Encapsulation is hide the values or state of a structured data object inside a class, preventing unauthorized parties' direct access to them.

Access Modifiers / specifiers are used in the Encapsulation.

Example :

The Access modifiers Public,private , Protected

Public in superclass are available/accessible to subclass and all other classes as well.

private are not directly accessible to subclass or any other class.

Protected in superclass are available/accessible to subclass and not to other classes.

Abstraction

The main purpose of abstraction is hiding the unnecessary details from the users, and only useful information show to the user.
In java the Abstraction is used two types;

  1. 1 : Abstract
  2. 2 : Interfaces

first use abstract keywords and make abstract function,classes,methods,properties.
Second is used interfaces(Pure Abstractions :- Hide all unnecessary information and only display useful information).We define interfaces with interface keywords.Interfaces has no constructors,and you cann't make the function/method in interfaces class.When you are make the derived class of interface you cann't use abstract keyword.In Interface you can use implements keywords.


Interfaces :
Multiple Inheritance is only possible on Interfaces.By using interfaces we get the two base class properties at the same time by using interfaces.
