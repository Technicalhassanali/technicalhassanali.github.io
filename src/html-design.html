Smooth Scroll

Add scroll-behavior: smooth to the <html> element to enable smooth scrolling for the whole page (note: it is also possible to add it to a specific element/scroll container 

html {
        scroll-behavior: smooth;
      }
Example:

You can see in the blow example we created two div and give the id and some styling when one section we click the other section open by using scroll-behavior:smooth; adding in css. 

<!DOCTYPE html>
    <html>
    <head>
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <style>

    html {
        scroll-behavior: smooth;
      }

    #section1 {
        height: 500px;
        background-color: #130f40;
        padding: 70px;
        color:white;
    }
    #section2 {
        padding: 70px;
        height: 500px;
        background-color: #f9ca24;
    }
    .btn{
    margin-left:50px;
    height:40px;
    line-height:40px;
    background-color:#f2f2f2;
    display:inline-block;
    cursor:pointer;
    }
    a{text-decoration:none;}
    </style>
    </head>
    <body>

<div id="section1">
  <h2>Section 1</h2>
  <p>Click on the link to see the "smooth" scrolling effect.</p>
  <a href="#section2" class="btn">Click Me to Smooth Scroll to Section 2 Below</a>
  <p>Note: Remove the scroll-behavior property to remove smooth scrolling.</p>
</div>


<div id="section2">
  <h2>Section 2</h2>
  <a href="#section1" class="btn">Click Me to Smooth Scroll to Section 1 Above</a>
</div>

    </body>
    </html>

ToolTip Design

a message which appears when a cursor is positioned over an icon, image, hyperlink, or other element in a graphical user interface.

<!DOCTYPE html>
        <html>
        <style>
        .tooltip {
          position: relative;
          display: inline-block;
          border-bottom: 1px solid black;
        }
        .tooltip .tooltiptext {
          visibility: hidden;
          width: 120px;
          background-color: black;
          color: #fff;
          text-align: center;
          border-radius: 6px;
          padding: 5px 0;

          /* Position the tooltip */
          position: absolute;
          z-index: 1;
        }


        .tooltip:hover .tooltiptext {
          visibility: visible;
        }

        </style>
        <body style="text-align:center;">

          <span class="tooltiptext">Tooltip text</span>
        </div>


        </body>
        </html>

Left & Right Position of Tooltip:

In Left Position you can change the right property to left and in right you can change left to right property in css. 

.tooltip .tooltiptext {
        top: -5px;
        right: 105%;  /* Change Position right : 105%; */
      }

Top & Bottom Position of Tooltip:

In Change the Position you can simply change the bottom,top property of css.

.tooltip .tooltiptext {
        width: 120px;
        bottom: 100%; /*In Top top: 100%; */
        left: 50%;
        margin-left: -60px;
        /* Use half of the width (120/2 = 60), to center the tooltip */
      }
