PHP cookie is a small piece of information which is stored at client browser. It is used to recognize the user. Cookie is created at server side and saved to client browser. Each time when client sends request to the server, cookie is embedded with request.

Create Cookies With PHP

setcookie(name, value, expire, path, domain, secure, httponly);

name : is assign the name of the cookie.

value : is the assign value to cookie like we assign the value to a variable $a = 10;

expire : when you set the cookie from user laptop then you can set the expire date / destroy of the cookie.If you not add the value then they not destroy at the time.They delete if they user clear the cached memory of the browser.
You have add the cookie expire time in second.

path : when you set the cookie and retrive the cookie set the path in slash / all the page or /page name is specific page can access the cookie.

domain : You can add the domain name like www.importantmcqs.com or sub-domain test.importantmcqs.com

secure : Accept two parameter Yes or No. Yes is used for the https Secure Connection then cookie set.

httponly: It mean that the cookie is access in local side or server side. If you pass the true then you only access the cookie in server side language (php,node.js) or NO is the client side you can access through JavaScript.
