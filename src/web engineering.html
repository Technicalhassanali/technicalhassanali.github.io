HTML

HTML stands for HyperText Markup Language.HTML is the standard markup language for documents designed to be displayed in a web browser.

Syntax : 

<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">

<title>Document</title>
</head>
<body>

</body>
</html>

<head> </head> | Head Tag

<head>tag contains metadata (document title, character set, styles, links, scripts), specific information about the web page that is not displayed to the user. Metadata provides browsers and search engines with technical information about the web page.

<head></head>

<title> </title> | Title Tag

<title>tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.It is most important for search engine optimization(SEO).

<title>Title Page </title>

<body> <body> | Body Tag

<body> ag defines the main content of an HTML document which displays on the browser. It can contain text content, paragraphs, headings, images, tables, links, videos, etc.

  function printTable() {
  let start = Number(document.getElementById("startN").value);
  let end  =  Number(document.getElementById("endN").value);
  let printvalue = document.getElementById("result");
  
  for (var i = 1; i <= end; i++) {

  }

Print Table using While Loop? 

<div>

</div>

<script>
function printTable() {
    let start = Number(document.getElementById("startN").value);
    let end = Number(document.getElementById("endN").value);
    let printvalue = document.getElementById("result");
    var i = 1;
    while (i <= end) {
    printvalue.innerHTML += (start + " * " + i + "=") + start * i + "<br>";
    i++;
    }
}
</script>
